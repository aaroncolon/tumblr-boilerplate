Tumblr Boilerplate

A fully functional bare bones Tumblr theme that works out of the box.

A template with the uncessary code stripped to ease the development process. Style it to your needs.

  • Download
  • Github

Features

Supports all post-types

All essential post-types (Image, Video, Text, Link, etc.) are semantically structured.

Localization Strings

Select variables appear in other Tumblr supported languages to communicate across countries.

Normalize CDN

The widely adopted CSS Normalize minified & hosted on a reliable delivery network.
